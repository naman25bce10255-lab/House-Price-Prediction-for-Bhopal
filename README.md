# Bhopal House Price Predictor

Project overview:

This project aims to provide an automated, data-driven solution for estimating real estate prices in Bhopal. By using Supervised Learning (Linear Regression), the tool analyzes property features to provide a market-aligned valuation for the year 2026.

Methodology:

The model follows a standard machines learning pipelines:

1. Data Collection: Focused on key Bhopal hubs like Arera Colony, Kolar Road, and Ayodhya Bypass
2. Feature Engineering: IMplemented a custom logic to account for BHK premiums (5% increase per extra room) to mimic real-market demand.
3. Algorithm: Linear Regression was chosen for its interpretability and efficiency with srtucted numercial data.

Conclusions:

The Bhopal House Price Predictor successfully demonstrated how AI can simplify complex urban problems, It provide Naman with a solid foundation in regression analysis and data preprocessing.
