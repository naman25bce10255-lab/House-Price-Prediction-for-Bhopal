# House-Price-Prediction-for-Bhopal

1. Project Overview
This project aims to provide an automated, data-driven solution for estimating real estate prices in Bhopal. By using Supervised Learning (Linear Regression), the tool analyzes property features to provide a market-aligned valuation for the year 2026.

2. Methodology
The model follows a standard machine learning pipeline:
Data Collection: Focused on key Bhopal hubs like Arera Colony, Kolar Road, and Ayodhya Bypass.
Feature Engineering: Implemented a custom logic to account for BHK premiums (5% increase per extra room) to mimic real-market demand.
Algorithm: Linear Regression was chosen for its interpretability and efficiency with structured numerical data.

3. Conclusion
The Bhopal House Price Predictor successfully demonstrates how AI can simplify complex urban problems. It provides Naman with a solid foundation in regression analysis and data preprocessing
